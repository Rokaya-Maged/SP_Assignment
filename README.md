# 💻 Structured Programming Assignment

## 📚 Multi-Problem C++ Console Application

---

## 👥 Team Members

This project was developed by a team of **4 members** as part of the **Structured Programming course**.

---

## 📌 Overview

This project is a **C++ console-based application** that solves multiple programming problems using fundamental concepts of structured programming such as:

* Functions
* Arrays & Strings
* Loops & Conditionals
* Recursion-like logic
* Matrices
* Basic algorithms

The program allows the user to select a problem and executes the corresponding solution.

---

## 🧩 Implemented Problems

### 🔢 Problem 1 – Prime Number Check

Checks whether a given number is **prime or not**.

---

### 🔤 Problem 2 – Matching Characters

* **Array version**: Compares two character arrays
* **String version**: Compares two strings
  Returns the number of matching characters from the beginning.

---

### 🔁 Problem 3 – Divisors of a Number

Displays all **positive divisors** of a given number in **descending order** and allows repeated execution.

---

### 🔄 Problem 4 – Shift Array Elements

Shifts elements of an array to the **right within a given range**.

---

### 🔢 Problem 5 – Matrix Multiplication

Performs multiplication of two **3×3 matrices**.

---

### 🧒 Problem 6 – Hanzalah Age Calculator

Calculates the age of **Hanzalah** based on a given month and year using specific conditions.

---

### 🔐 Problem 7 – Encryption & Decryption

* Encrypts a message by shifting characters (+5 ASCII)
* Decrypts a message by reversing the process

---

### 🌐 Problem 8 – Graph Matrix

Converts an input representation into a **6×6 adjacency matrix**.
